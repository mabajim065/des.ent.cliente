Ejercicio 5: Unir Array en una Cadena (Objeto Array)

Instrucción: Escriba un programa JavaScript simple para unir todos los elementos de una matriz en una cadena.

Revisa la Consola del Navegador (F12)

La solución utiliza el método **.join()**, parte del **Objeto predefinido Array**, que convierte todos los elementos del array en una sola cadena, separados por un delimitador (por defecto, una coma).
